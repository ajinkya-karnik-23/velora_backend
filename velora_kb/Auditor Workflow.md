# Auditor Workflow

Auditors do not create controls.

They gather evidence demonstrating that a control was executed.

Typical workflow:

Control
→ Request Evidence
→ Receive Artifacts
→ Validate Completeness
→ Execute Testing
→ Reach Conclusion

Evidence is commonly obtained from:

- SAP
- ServiceNow
- Jira
- SharePoint
- Outlook
- Excel trackers
- Internal reporting systems
